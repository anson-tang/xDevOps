About this program
==================

xstardict is a stardict package built with python.

Deploy a stardict with dictionary on ubuntu.

options:
----------

Use -i to install stardict.

Use -d to install the offline dictionary.

***

How to use
==================

./install.py -h

OR

python2.7 install.py -h
